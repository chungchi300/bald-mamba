# what is it for

An opinionated Expo Starter for more heavy **redux** usage in order to build complex mobile app in a more maintainable way.

# Features

## Basic

- Connected react router
- Redux
- Redux Api Middleware
- Redux Thunk
- Redux Form
- React Native Paper
- react-native-side-menu
